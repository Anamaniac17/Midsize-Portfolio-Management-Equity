# Stockmarket- Portfolio-Management #
From market theory to real returns —Built a real-time stock portfolio and decoded market movements using data, fundamentals, and technical indicators.
Project Overview

This project provides a comprehensive study of the Indian stock market, including its historical development, regulatory framework (SEBI), market structure (NSE & BSE), and key trading concepts. It combines theoretical understanding with practical investment analysis. 


Key Components

Overview of Indian stock market evolution and major milestones

Role, objectives, and functions of SEBI

Market structure, trading procedures, and key terminology

Fundamental analysis (financial ratios, company evaluation)

Technical analysis (trend analysis, moving averages, RSI, support & resistance)

Case study of a portfolio built using ₹50,000 across selected stocks

Case Study Outcome

Tracked 5 stocks over 20 trading days

Portfolio value grew from ₹56,185 to ₹62,967

Generated a 12% return through diversified investment and analysis. 


Tools Used

Microsoft Excel

Market data analysis

Financial ratio and performance evaluation
